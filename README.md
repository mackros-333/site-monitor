# Портфолио: Мониторинг доступности сайтов

Разработчик: Смирнов Илья | Python, автоматизация, логирование

---

## Что это за проект

Автоматический мониторинг сайтов. Проверяет список URL, фиксирует статус (OK/FAIL) и записывает результат в лог-файл.

Проект создан в рамках портфолио.

---

## Что проект умеет

- Проверяет доступность сайтов (HEAD-запрос, быстрее GET)
- Определяет статус: OK (200) или FAIL (ошибка соединения)
- Записывает результат в лог-файл с датой и временем
- Лог читается как в консоли, так и в текстовом файле
- Легко добавить свои сайты для проверки
